<Steps>
<Step title="Clone Starter Repo">
Run the following from your terminal to clone the starter repo with the sequence-cli:

<CodeGroup>
```bash React
npx sequence-cli boilerplates create-embedded-wallet-react-starter
```

```bash Next
npx sequence-cli boilerplates create-kit-embedded-wallet-nextjs-starter
```
</CodeGroup>

Selecting the test keys option will deploy a local of Sequence Kit to `localhost:4444` with complete with social, email and web3 login along with common example functionality.


TO DO: (REFERENCE BUILDER)
</Step>
<Step title="Copy example .env file">

```bash
cd embedded-wallet-react-boilerplate && cp .env.example .env
```
</Step>
<Step title="Install Packages and start app">

<CodeGroup>
```bash pnpm
pnpm install && pnpm dev
```

```bash npm
npm install && npm run dev
```

```bash yarn
yarn install && yarn dev
```
</CodeGroup>

</Step>
<Step title="Connect and Authenticate">
Navigate to `localhost:4444` and click connect to try out the authentication.
</Step>
<Step title="Use Sequence!">

Now that you have a working authentication, you can take look at our examples for the most common functions:
- [Sign messages](/solutions/wallets/embedded-wallet/examples/use-wallets#sign-messages)
- [Send transactions](/solutions/wallets/embedded-wallet/examples/use-wallets#send-transactions)
- [Call contracts](/solutions/wallets/embedded-wallet/examples/use-wallets#call-any-contract)
- [Verify a user on your backend](/solutions/wallets/embedded-wallet/examples/verification#prerequisite-deployment-of-a-parent-wallet)
</Step>
</Steps>


<Tip>
We are using example variables in the .env file, so we recommend creating a project on [Sequence Builder](https://sequence.build/project/default/wallet/embedded-wallet) and [setting up your own wallet configuration](/solutions/builder/embedded-wallet) and update the environment variables to use more social providers for authentication, customize email templates, and more.
</Tip>
